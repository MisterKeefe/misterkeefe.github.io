Labs

A list of small projects I've worked on that are somewhere between experiments and useful software.

  • Victorian Chapter Heading Generator -- This one is kind of hard to describe. It generates old Victorian-style chapter headings.
  • dom-particles — An npm library for particle effects in the DOM.
  • pytest-involve — A pytest plugin for specifying the files you want tested, not the tests you want to run.
  • pixel-factory -- A tool for making "programmer art" by writing tiny JavaScript programs. It's worth reading the manual.
